# ==============================================
# ИНСТРУКЦИИ ПО НАСТРОЙКЕ
# ==============================================

# 1. ЯНДЕКС.МЕТРИКА
# - Зарегистрируйтесь на https://metrika.yandex.ru
# - Создайте счетчик или используйте существующий
# - ID счетчика найдете в URL: https://metrika.yandex.ru/dashboard?id=XXXXXXXX
# - Для получения OAuth токена:
#   https://oauth.yandex.ru/authorize?response_type=token&client_id=YOUR_CLIENT_ID
# - Права: metrika:read

# 2. INSTAGRAM GRAPH API
# - Создайте приложение на https://developers.facebook.com
# - Подключите Instagram Business Account
# - Получите долгоживущий Access Token (60 дней)
# - Необходимые права: instagram_basic, instagram_manage_insights
# - Инструкция: https://developers.facebook.com/docs/instagram-api/getting-started

# 3. ВКОНТАКТЕ API
# - Создайте приложение: https://vk.com/apps?act=manage
# - Получите Access Token
# - Необходимые права: stats, groups, wall, stories (опционально: ads)
# - ID группы найдете в URL: https://vk.com/club12345678 -> 12345678

# 4. TELEGRAM BOT API
# - Откройте Telegram и найдите @BotFather
# - Отправьте команду /newbot
# - Следуйте инструкциям для создания бота
# - Скопируйте полученный токен
# - Для работы с каналом: добавьте бота в администраторы канала

# 5. 1С API
# - Настройте HTTP-сервис в вашей конфигурации 1С
# - Создайте endpoint для получения данных о продажах
# - Если используете OData - укажите URL базы
# - Настройте аутентификацию (Basic Auth или Token)

# ==============================================
# ТЕСТИРОВАНИЕ
